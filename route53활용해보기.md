```
간단한 클라우드 환경을 구축해보자

- route53 : 도메인을 통해 내 서비스에 접근, 네임서버를 통해 도메인과 ip 연결
- vpc : 가상의 네트워크 구축
- igw : 외부 인터넷으로부터 진,출입
- lb : (당장 필요하지 않지만) 오토 스케일링 등으로 유동적으로 변하는 인스턴스에 고정적인 접근을 제공 및 부하 분산
- nacl : 서브넷 방화벽
- sc : 인스턴스 방화벽
```
